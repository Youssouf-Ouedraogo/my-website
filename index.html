A* Algorithm Implementation(Python, Pycharm)

Given a maze, a starting point and a goal, find the shortest path from the starting point to the goal. I achieved that using repeated forward A* and repeated backward A*. Both are the same except their execution. for repeated forward A* the robot will learn the wall as it goes. However, for repeated backward A*, the robot has the full path before moving. A* works as the following: we compute the shortest path from where we are to our destination. Without taking into account the walls we have not encounter. We start following the path until we hit a wall. We add that wall in our list of walls. we compute the shortest path where we are standing now to our destination. This time however, we eliminate the possibility of passing through the wall. We repeat this process until we get to our goal just like with the animation below
